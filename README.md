#1. 라프텔 파티 개요
   
   라프텔의 애니메이션을 동시시청 가능하게 만드는 프로그램입니다.
   라프텔 가입 및 ngrok 개인 서버가 필요합니다. 또한 기초적인 cmd 명령어 숙지도 필요합니다.
   
#2. 라프텔 파티ver.1(방 제목, 방 비밀번호) 사용법
   
   1️⃣cmd를 2개 엽니다.
     한 개는 node 명령어를 이용해 포트 하나를 개방하고
     다른 하나는 ngrok으로 node 명령어로 개방한 포트를 이용해 서버를 엽니다.
   
   2️⃣서버를 연 방장은 라프텔의 애니메이션 페이지를 열고 확장프로그램을 클릭하고 방주소 입력칸에 ngrok에서 연 서버 주소를 입력하고 방 제목, 방 비밀번호를 작성한 뒤 방 생성 버튼을 누릅니다.

   3️⃣방 참가자들은 방장이 공유한 서버 주소를 입력하고 방 제목 방 비밀번호를 입력한 뒤 방 참가 버튼을 입력합니다.

   *️⃣방장은 cmd를 통해 방에 접속한 사람을 알 수 있습니다. 방에 들어가고 동기화가 잘 되는 지 확인하시면 됩니다.
     방을 참가한 상태에서 확장프로그램을 누르면 방장이 보고 있는 애니메이션의 제목, 에피소드, 동영상 시간등이 표기됩니다.
     나가고 싶다면 확장프로그램을 누르고 방 나가기 버튼을 클릭하면 됩니다.
   
#3. 라프텔 파티ver.2(링크) 사용법
 
   1️⃣cmd를 2개 엽니다.
     한 개는 node 명령어를 이용해 포트 하나를 개방하고
     다른 하나는 ngrok으로 node 명령어로 개방한 포트를 이용해 서버를 엽니다.
   
   2️⃣서버를 연 방장은 라프텔의 애니메이션 페이지를 열고 확장프로그램을 클릭하고 방주소 입력칸에 ngrok에서 연 서버 주소를 입력한 뒤 방 생성 버튼을 누릅니다.

   3️⃣방 참가자들은 방장이 공유한 서버 주소를 주소창에 붙여넣기만 하면 됩니다.

   *️⃣방장은 cmd를 통해 방에 접속한 사람을 알 수 있습니다. 방에 들어가고 동기화가 잘 되는 지 확인하시면 됩니다.
     방을 참가한 상태에서 확장프로그램을 누르면 방장이 보고 있는 애니메이션의 제목, 에피소드, 동영상 시간등이 표기됩니다.
     나가고 싶다면 확장프로그램을 누르고 방 나가기 버튼을 클릭하면 됩니다.


#4. 버전 차이

   ver.1은 방 제목과 비밀번호를 요구하기에 우연하게 사람이 주소를 알아내도 들어오기 어렵습니다.(보안성 위주)
   ver.2는 간편하지만 주소가 유출되었을 시 입장한 사람 관리가 불가능합니다.
